# EngSub — 看 YouTube，顺便把英语学了

**把你爱看的视频，变成你的英语课堂**

边看字幕边查词，遇到生词一键收藏，复习巩固不再靠死记硬背。

[EngSub 官网](https://tube-learning-offical.vercel.app/)

## 这是什么？

**EngSub** 是一款面向英语学习者的 YouTube 沉浸式学习工具。

不需要另找教材，不需要脱离真实语境——直接打开你本来就想看的视频，在**英文字幕**里点词查义、收藏生词、反复复习。电影、音乐、访谈、纪录片……兴趣在哪，学习就在哪。

## 能帮你做什么？

| 痛点 | EngSub 怎么解决 |
|------|----------------|
| 看英文视频，遇到生词要切出去查 | 长按 / 双击字幕单词，**原地查词典** |
| 查完就忘，没有积累 | 一键**收藏生词**，自动同步到云端 |
| 不知道看什么、从哪里开始 | 首页精选**教育 / 影视 / 音乐 / 节目**推荐 |
| 字幕加载失败 | 支持从 **DOWNSUB** 导入 SRT 英文字幕 |
| 只想复习之前学过的词 | **单词列表 + 卡片式复习**，随机抽 20 词巩固 |

## 核心功能

### 一键导入 YouTube 视频

- 在 YouTube App 里**分享视频到 EngSub**
- 或在 EngSub 底部**粘贴视频链接**，点击「开始」
- 自动记录**播放历史**，随时继续上次的进度

### 互动式英文字幕

![EngSub 视频字幕同步学习界面](/flutter_02.png)

- 自动加载视频英文字幕，字幕随播放**高亮滚动**
- **单击**某行字幕 → 视频跳转到对应位置
- **长按 / 双击**单词 → 弹出词典释义
- 支持拖动视频与字幕之间的分隔条，自由调整布局

### 三种查词模式

- **英英词典** — 用英语理解英语，适合进阶学习者
- **汉英词典** — 快速看懂中文释义
- **Google 翻译** — 短语、句子也能查

查过的结果会**本地缓存**，下次打开更快。

### 生词收藏与复习

- 查词时点击星标，把单词加入**个人词库**
- Google 登录后，词库**云端同步**，换设备也不丢
- **单词列表**：浏览、左滑删除、分页加载全部收藏
- **单词复习**：卡片式随机抽词，翻页巩固记忆


### 更多贴心设计

- 中英文界面切换
- 浅色 / 深色 / 跟随系统主题
- 横屏**影院模式**，大屏沉浸观看

## 3 步开始


1. 打开 EngSub
2. 分享或粘贴一个 YouTube 视频链接
3. 看字幕 → 点词查义 → 收藏生词 → 定期复习


> **小提示**：第一次使用可在首页查看「如何使用」说明；若字幕无法加载，可尝试从 DOWNSUB 导入英语 SRT 字幕文件。
